# CodeAlpha Web Development Internship

Welcome to my **CodeAlpha Web Development Internship** repository. This repository contains the web development projects I completed as part of the internship program. These projects helped me strengthen my understanding of front-end web development by applying concepts of HTML, CSS, and JavaScript to build responsive and interactive websites.

# 📌 About

This repository showcases the practical tasks assigned during the CodeAlpha Web Development Internship. Each project focuses on different aspects of web development, including responsive layouts, user interaction, and modern UI design.

# 📋 Completed Tasks

## ✅ Task 1: Calculator

### Description

A responsive web-based calculator capable of performing basic arithmetic operations such as:

* Addition
* Subtraction
* Multiplication
* Division

The application provides a clean and intuitive user interface for quick calculations.

### Features

* Responsive design
* Interactive calculator buttons
* Real-time calculations
* Simple and modern UI

### Technologies Used

* HTML5
* CSS3
* JavaScript

## ✅ Task 2: Image Gallery

### Description

A responsive image gallery developed using HTML and CSS that displays images in an attractive grid layout. The design adapts seamlessly across desktops, tablets, and mobile devices.

### Features

* Responsive image grid
* Modern gallery layout
* Hover effects
* Clean user interface
* Mobile-friendly design

### Technologies Used

* HTML5
* CSS3
